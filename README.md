# Carbon-Aware Marine AI for Sustainable SST Forecasting

This repository is the static GitHub Pages showcase for a South Florida Atlantic Coast next-day sea surface temperature forecasting case study. It compares evaluated models, summarizes the NOAA OISST data workflow, and carefully distinguishes CodeCarbon estimates and cloud-region simulations from directly measured physical emissions.

The showcase contains INFO, RESEARCH QUESTION, STUDY AREA AND DATA, FORECASTING WORKFLOW, MODEL COMPARISON, RESULTS, CARBON-AWARE EXECUTION, LIMITATIONS, and PROJECT LINKS.

## Repository structure

- `index.html` - semantic showcase page
- `style.css` - DCGAN-reference glassmorphism styling
- `script.js` - neural background, theme toggle, and scroll-to-top behavior
- `images/` - verified workflow, model, forecast, and regional-comparison figures plus reference font and favicon

## Disclaimer

The regional emissions comparison is modelled from a measured-workload estimate and published regional carbon data. It is not a direct measurement in every region and does not establish universal carbon savings. The supplied paper is treated as a course research report, not as a publication.
